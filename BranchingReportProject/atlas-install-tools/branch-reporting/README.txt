branch-reporting shows status of latest build for specified branch across repositories

1. requirements
- Python 3.6
- pip

2. how to install

run: pip install -r requirements.txt 

3. how to run 

 - option 1 - ./report-branch-status.py --j jenkins.corp.ad.ctc -u first.lastname -t <YOUR JENKINS TOKEN> -r repolist.txt -d Digital/CTC/ATLAS -b develop
